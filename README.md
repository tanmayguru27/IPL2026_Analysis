# IPL2026_Analysis
This project presents an end-to-end analysis of the IPL 2026 season using Python, Pandas, and Matplotlib. The dataset was cleaned by handling missing values, removing duplicates, and converting date fields into the appropriate format. Exploratory data analysis was performed to evaluate team performances, win percentages, toss outcomes, match margins, venues, and host cities. Multiple visualizations, including bar charts, pie charts, line graphs, horizontal bar charts, and scatter plots, were created to uncover meaningful patterns and trends. The project demonstrates practical data cleaning, manipulation, analysis, and visualization skills while showcasing the use of Pandas and Matplotlib to generate actionable insights from real-world sports data.
